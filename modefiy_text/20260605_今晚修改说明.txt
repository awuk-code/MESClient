一、扫码过站限制调整

涉及文件：
- ui/processstationleftpanel.h
- ui/processstationleftpanel.cpp

涉及函数 / 类：
- ResumeReasonDialog
- ProcessStationLeftPanel::onExecuteBtnClicked
- ProcessStationLeftPanel::onPauseBtnClicked
- ProcessStationLeftPanel::onResumeBtnClicked
- ProcessStationLeftPanel::handleScanInAction

- ProcessStationLeftPanel::handleNgResult
- ProcessStationLeftPanel::handlePauseResult
- ProcessStationLeftPanel::isProductPassed
- ProcessStationLeftPanel::isProductPaused
- ProcessStationLeftPanel::validateProductNotPassed
- ProcessStationLeftPanel::validateProductNotPaused
- ProcessStationLeftPanel::isStatusMarked

1. PASS 操作增加确认弹窗。
   选中 PASS 后点击执行，会先提示确认该产品是否通过当前工序，确认后才写入 PASS 状态。

2. 暂停状态下禁止 PASS / NG。
   产品处于暂停状态时，再执行 PASS 或 NG，会提示“请先解除暂停。”。

3. 解除暂停增加理由登记。
   点击解除暂停时，会弹出解除暂停原因登记窗口，必须填写理由后才会取消暂停状态。

4. PASS 后禁止继续操作。
   产品已 PASS 后，不允许重复 PASS、NG、暂停、解除暂停，也不允许重复扫入继续操作。

5. PASS / NG 的产品来源调整。
   PASS 和 NG 不再读取状态表当前选中行，而是读取输入框中的产品 SN，再到状态信息表中查找对应行。
   如果表格中没有该 SN，会提示需要先扫入该产品 SN。

6. 暂停 / 解除暂停的产品来源保持不变。
   暂停和解除暂停仍然针对状态信息表中当前选中的具体行操作。

7. 扫入后不再清空输入框。
   这样用户扫入 SN 后，可以先去右侧完成物料、工艺、上传等操作，回来选择 PASS 或 NG 时继续使用输入框中的 SN。

1. 曾临时实现过导出前高级筛选窗口。
   支持按字段关键字、日期范围等条件筛选后再导出。

2. 经确认，原导出逻辑本身已经导出当前页面经过用户筛选后的结果。
   例如状态筛选、顶部搜索、表头日期筛选等，都会先作用在当前表格代理模型上，导出时拿到的就是筛选后的表格数据。

3. 因此正式代码中撤销了导出前筛选弹窗，恢复为导出当前页面当前筛选结果。

4. 高级筛选导出的代码备份保留在项目目录：

三、表格 TabBar 字体显示调整

涉及文件：
- res/style/style.qss
- ui/basepagewidget.cpp

涉及位置 / 函数：
- QTabBar#BasePageTabBar::tab
- BasePageWidget::initUI

1. 表格页签 BasePageTabBar 的文字字号改为与正文一致。
   当前统一为 13px。

2. 修复 TabBar 中文文字被裁剪、缺笔画的问题。
   原因是 Qt5 的 QTabBar 在 QSS 设置字号后，内部 tabSizeHint 与实际绘制字号不稳定。

3. 最终处理方式：
   - QSS 中给 BasePageTabBar::tab 设置固定 height。
   - 在 BasePageWidget 中给 QTabBar 明确设置 13px 字体。

1. 左侧扫码过站扫入产品 SN 成功后，会发出 productSnScanned 信号。

2. ProcessStationPage 作为左右面板的整体容器，负责关联左侧信号和右侧槽函数。
   这样左侧和右侧不直接互相依赖，左右联动逻辑集中在大页面中维护。

3. 右侧 ProcessStationRightPanel 接收到 SN 后，会将 SN 添加到产品 SN 下拉框中。
   如果该 SN 已存在，则不重复添加，只切换为当前选中项。

4. 右侧当前产品 SN 会同步影响资料上传页。
   DataUpload 页面中的 snXXXXX_DOC / snXXXXX_IMG 会自动更新为当前扫入产品对应的 SN_DOC / SN_IMG。

5. 右侧有产品 SN 字段的表格页会按照当前 SN 进行本地过滤。
   当前支持：
   - 工序物料信息：字段 productSN
   - 工具设备：字段 productSN
   - 更换物料信息：字段 productSNCode

6. 切换任务或清空工序站数据时，会清空右侧产品 SN 下拉框，避免上一个任务的 SN 残留到新任务中。
